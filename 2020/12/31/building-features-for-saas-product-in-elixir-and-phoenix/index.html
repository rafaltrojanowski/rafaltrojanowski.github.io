Building features for SaaS product in Elixir and Phoenix

Dec 31, 2020

The inspiration to start this side-project came out from few facts:

  1. I have been inspired by what Chris Olivier makes for Ruby community. Expecially after listening to this podcast that I found on indiehackers.com: https://www.indiehackers.com/podcast/183-chris-oliver-of-gorails
    While back I have seen a Jumpstart Rails Project: https://jumpstartrails.com/
    which was something interesting, and I was wonderning how it has been build.
  2. I’ve read this blog post: https://devblast.com/b/why-I-rebuilt-my-website-from-scratch-with-elixir-and-not-ruby
    In a nutshell the author says that we need to keep learnig new stuff or we stagnate. I totally agree with it. That’s why I decided to go with Elixir, instead of Ruby I don’t regret it. There were the moments of short frustration when I could not build something that fast as before, but overall it was interesting experience and learning opportunity that requires mindset shift - because Elixir is functional.
  3. ElixirCasts (https://elixircasts.io/)
    I bought an access a while ago, I wanted wrap up the knowledge that I got from the courses.

The good

For the most boring parts that are required for almost every app I could find generators. I'm talking about phx_gen_auth and Torch. Thanks to that I could see new features quickly and I felt motivated to proceed.

The rest of the app was a fully customized and I had to reverse-engineer other apps, go through a few of blog posts or just simply hack something on my own and look for help in Github issues - standard. There is also Plausible repo on Github which comes handy when I want to check some implementation details. That’s how the inspiration for Billing.active_subscription_for(user_id) function came.

What's new

Elixir is quite fast. VSCode has a nice support for it! There is a nice podcast that I like to listen after coding called Thinking Elixir. I also joined Slack group and what's good I can see that there are new job offers popping out regullary so that's good trend. I also see that people are willing to help for those who got stuck with something and there is dedicated #beginners channel. That's pretty nice. There is also Elixir track on Exercism, and the hard quizes are really challanging but, this is probably topic for another post. What I found also interesting is the whole serie on how to build a Tetris with LiveView which is available on Groxio's Youtube channel: https://www.youtube.com/watch?v=8h8tWcz115M&list=PLKBMoE8mCkXi-sAkesjaUnDQqyrkAK8R5
